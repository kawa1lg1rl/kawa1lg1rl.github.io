## 군대에서 개발
군대에서 개발 할 수 있을까 라는 생각에서 검색을 해보았다.  
입대 전 최대한의 준비를 해놓고 입대하려는 생각에 일단 글을 적어본다.  

### Google Cloud
Google Cloud Platform에는 VM을 돌릴 수 있게 서비스를 제공한다.  
VM인스턴스를 생성하고 SSH로 접속을 할 수 있다.
또한 브라우저상으로 ssh 접속을 할 수 있도록 지원해주니 편하게 쓸 수 있을거같다. 될진 모르겠지만

안드 앱개발의 경우에는 아래와같이 command line으로도 가능하다.  
https://medium.com/@authmane512/how-to-build-an-apk-from-command-line-without-ide-7260e1e22676

VCPU 1, RAM 6.5GB, Disk 10GB로 인스턴스 생성시 한달 예상 40달러 정도인 것 같다.

### Goorm IDE
구름 IDE를 사용해서 개발을 할 수 있다는 글을 여럿 봤다.  
하지만 환경이 좋지않다 구름 IDE조차 사용하기 불편할 거라는 얘기가 많이 있다.  
하지만 가능은 하다는 것 같으니 써봐도 괜찮을 것 같다.

추가로 Android 개발은 어떻게 하는지, 가능한지 모르겠다. Goorm에 문의 넣어놨으니 답변 오면 수정해야겠다.

### 노가다
검색을 하다보니 네이버에 프로젝트를 올려놓고 싸지방에 들어갈때마다 받아서 개발했다는 글을 보았다. 또 어떤 글에서는 네이버클라우드를 사용하면 로그남고 불려간다는데 미필이라 잘 모르겠다.
